# Verificador de Dependências Funcionais

Este código foi criado com a intenção de atender as exigências de um trabalho para a disciplina de Banco de Dados, cursada em 2026 durante o segundo ano do ensino médio técnico integrado em informática para a internet na UTF-PR. 
Ele buscar realizar a **verificação automática de dependências funcionais** em qualquer tabela de um banco de dados **PostgreSQL**, com base nos registros existentes.

---

## Instruções de Execução

1. **Instalar dependências (apenas na primeira execução):**  
   No terminal, dentro da pasta do projeto, execute:
   
   ```bash
   npm install pg

3. **Executar o verificador:**  
   Após a instalação, rode o comando:
   
   ```bash
   node veficador_dependencias.js

5. **Visualizar os resultados:**  
   As dependências funcionais identificadas serão exibidas diretamente no terminal.
